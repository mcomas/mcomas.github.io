Marc Comas-Cufí

Associate professor in Statistics

Universitat de Girona

I am a member of the Statistics and Compositional Data Analysis research group at the University of Girona. My primary interests include Compositional Data modeling, Multivariate Count Data analysis, Clustering methods, Survival analysis, Competing Risk analysis, and Causal inference. I enjoy programming and exploring new languages for diverse applications, primarily using R and Python for statistical analysis and data management, Java for GUI development, and JavaScript for web programming.

I have experience with various languages such as C/C++, Fortran, Visual Basic, VBA, Stata, Matlab, Maple, and Lisp. Additionally, I am proficient with common markup languages like HTML, LaTeX, Markdown, XML, and JSON.

In my daily work, I prioritize open-source tools, frequently utilizing RStudio, GNU Make, and Git to enhance productivity and collaboration.
